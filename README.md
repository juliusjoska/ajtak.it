# ajtak.it -- IT Services & Software Development

[![Website](https://img.shields.io/badge/Website-ajtak.it-000000?style=flat&logo=googlechrome&logoColor=white)](https://ajtak.it)
[![GitHub](https://img.shields.io/badge/GitHub-juliusjoska-181717?logo=github&logoColor=white)](https://github.com/juliusjoska)

Independent software development and IT services based in the Czech Republic. Building web applications, mobile apps, SaaS products, and managing self-hosted infrastructure.

---

## Services

**Web Development** -- Custom websites and web applications built with Next.js, React, and TypeScript. From marketing landing pages to full-stack SaaS platforms.

**Mobile Development** -- Cross-platform mobile applications with Flutter/Dart. Native performance on iOS and Android from a single codebase.

**SaaS Products** -- End-to-end product development including architecture, database design, authentication, payments, and deployment.

**Infrastructure & DevOps** -- Self-hosted infrastructure management on Proxmox. Docker orchestration, monitoring, backups, CI/CD pipelines, and security hardening.

**IT Consulting** -- Technical architecture reviews, technology stack selection, and infrastructure planning for small and medium businesses.
